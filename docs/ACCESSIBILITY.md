# Accessibility Statement & Gestures

We strive for accessibility:
- ARIA roles on key controls (todo expand)
- Keyboard navigation: focusable buttons, Enter to activate hex selection
- Contrast: Tailwind color choices ensure WCAG AA where possible
- Large touch targets (>44px) for mobile
- Touch gestures:
  - Tap hex to select
  - Drag hex to move
  - Pinch-to-zoom: Optional (requires additional implementation)
  - Long-press context menu: Optional

Please run axe / Lighthouse to validate.
